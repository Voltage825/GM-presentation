Front-End Experience

Celine focuses on defining a specification on how components can interact with each other through a micro-frontend framework. While decomposing the backend into microservices, we often leave the UI a a large monolithic app which leaves the same problems as our old backend designs.

Celine addresses these issues by focusing on component and page based separation, each with its own http server, servicing each individual component. Allowing independant development, lifecycle and deployment models.
